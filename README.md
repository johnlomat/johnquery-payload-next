## Features

- **Dynamic Content Management**: Integrated with Payload CMS 3.x for managing and delivering dynamic content.
- **PostgreSQL Database**: Reliable and scalable relational database for storing CMS data.
- **Cloud Media Storage**: Cloudinary integration for optimized image and media delivery.
- **Modern Rich Text Editor**: Lexical editor for advanced content editing capabilities.
- **ISR Rendering**: Utilizes Next.js Incremental Static Regeneration (ISR) to update static content without rebuilding the entire site.
- **Tailwind CSS**: Provides a utility-first CSS framework for fast and responsive design.
- **React 19**: Uses the latest React version for interactive user interfaces and state management.
- **Flowbite React**: Incorporates Flowbite React component library for pre-built UI components.
- **Next.js App Router**: Uses Next.js 15 App Router for dynamic routing and page navigation.
- **Responsive Design**: Ensures optimal viewing experience across a wide range of devices.
- **Custom Components**: Includes reusable components for headers, footers, sections, and UI elements.
- **SEO Optimization**: Enhances visibility with optimized metadata and page titles.

## Technologies Used

- **Next.js 15**: React framework for server-side rendering, client-side routing, and ISR.
- **Payload CMS 3.x**: Headless CMS for managing and delivering dynamic content.
- **PostgreSQL**: Relational database for storing CMS data.
- **Cloudinary**: Cloud-based media storage and optimization.
- **Lexical Editor**: Modern rich text editor for content management.
- **React 19**: JavaScript library for building user interfaces and managing state.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Flowbite React**: Component library for pre-built UI components integrated with Tailwind CSS.
- **TypeScript**: Type-safe JavaScript for improved developer experience.
- **Vercel**: Hosting platform for static websites and projects, providing seamless deployment and hosting.
- **Node.js**: Runtime environment for server-side logic and API handling.
- **Nodemailer**: Node.js module for sending emails from the contact form.

## Deployment

The website is deployed using Vercel, ensuring seamless deployment and hosting with automatic deployments from GitHub.
